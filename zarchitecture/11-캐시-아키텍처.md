# 11. 캐시 아키텍처

> **범위:** tcf-cache, EhCache 3, Spring Cache, OM Cache Admin  
> **관련:** [zman/16-Cache구조.md](../zman/16-Cache구조.md) · [zdoc/캐시관리.md](../zdoc/캐시관리.md)

---

## 1. 개요

tcf-cache는 Spring Cache + **EhCache 3 (JCache)** 기반 공통 캐시 환경을 제공한다.

| 항목 | 값 |
|------|-----|
| 모듈 | tcf-cache (JAR) |
| 주 사용 | tcf-om (공통코드, Catalog) |
| 비활성화 | `nsight.tcf.cache.enabled: false` |

---

## 2. 아키텍처

```mermaid
flowchart LR
    OM[tcf-om Service] -->|@Cacheable| SC[Spring Cache]
    SC --> EC[EhCache 3 JCache]
    EC --> R1[commonCode]
    EC --> R2[serviceCatalog]
    EC --> R3[sessionRegion]
    ADMIN[OM Cache Admin] -->|OmCacheHandler| SC
```

---

## 3. 캐시 Region

| alias | 상수 | 용도 | TTL |
|-------|------|------|-----|
| commonCode | TcfCacheNames.COMMON_CODE | 공통코드 | 30분 |
| serviceCatalog | TcfCacheNames.SERVICE_CATALOG | ServiceId Catalog | 60분 |
| sessionRegion | TcfCacheNames.SESSION_REGION | 세션/리전 | 10분 |

설정: `classpath:ehcache.xml` (JAR 내장)

---

## 4. 사용 패턴

### 4.1 선언적

```java
@Cacheable(cacheNames = TcfCacheNames.COMMON_CODE, key = "#codeGroup")
public List<Map<String, Object>> findByGroup(String codeGroup) { ... }

@CacheEvict(cacheNames = TcfCacheNames.COMMON_CODE, allEntries = true)
public void save(...) { ... }
```

### 4.2 프로그램

`TcfCacheSupport` — evict(), evictAll(), snapshotEntries()

---

## 5. STF·거래통제 연계

Catalog·TC 조회 시 Cache hit → DB 부하 감소.

**주의:** Catalog/TC 변경 시 **CacheEvict** 필수 — stale Allow-List 방지.

---

## 6. OM Cache Admin

| URL | serviceId |
|-----|-----------|
| /om/admin/cache.html | OM.Cache.* |

- bootRun: http://localhost:8099/om/admin/cache.html
- Region snapshot, evict

```gradle
implementation project(':tcf-cache')   // tcf-om
```
