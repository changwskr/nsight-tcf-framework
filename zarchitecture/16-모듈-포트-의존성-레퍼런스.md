## 2. 의존성 그래프

```
tcf-util
  └── tcf-core
        └── tcf-web
              ├── tcf-cache (tcf-om)
              ├── tcf-eai (ic, sv)
              ├── ic-service … mg-service
              ├── tcf-om
              ├── tcf-jwt
              └── tcf-batch

독립 실행:
  tcf-gateway  (HTTP → downstream)
  tcf-ui       (Relay HTTP)
  tcf-uj       (Relay → Gateway)
```

## 9. 빠른 기동 명령

```bash
# 단일 업무
gradle :sv-service:bootRun
tcf-scripts/run-local.bat sv

# Gateway UI 전체
gradle :tcf-om:bootRun
gradle :tcf-gateway:bootRun
gradle :tcf-uj:bootRun

# ztomcat
ztomcat/deploy-wars.bat all && ztomcat/start.bat
```
