처음이신가요? — 추천 순서

아래 4단계만 따라가면 CAP 산정부터 ENV 점검까지 이어서 진행할 수 있습니다.

  1. 조건·시나리오 입력 지점, 사용자, TPS 시나리오, VM 규격을 입력합니다. CAP 용량 산정 →
  2. 산정 결과 확인 TPS, AP 대수, maxThreads, Hikari poolSize를 단계별로 확인합니다. 결과 탭 (동일 화면) →
  3. 환경설정 연동 ENV-002에서 산정을 실행하고 ENV-003·004에서 계층별 값을 점검합니다. ENV-002 →
  4. 종합·Rule 점검 종합 보고서와 SC Rule로 Timeout·Pool 위험을 최종 검토합니다. 종합 보고서 →
