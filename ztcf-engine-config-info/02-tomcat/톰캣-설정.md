## 구조

```text
02-tomcat/
├── bin/setenv.sh              # JAVA 21, jvmRoute, Heap
├── conf/
│   ├── server.xml             # Connector + Cluster
│   ├── context.xml            # JSESSIONID, Manager pathname
│   └── web.xml
├── scripts/check-tomcat.sh
└── systemd/tomcat-nsight.service
```

## 필수

- 노드별 `JVM_ROUTE` 유일 · Apache `route`와 동일  
- 각 WAR `<distributable/>`  
- 센터 간 DeltaManager 미적용  

```bash
./scripts/check-tomcat.sh
$CATALINA_HOME/bin/catalina.sh configtest
```
