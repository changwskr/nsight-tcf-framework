## 1. 종합 판정

| 판정 | 내용 |
|------|------|
| **정합** | WAR·Context·JDK 21·Proxy·health 경로가 프로젝트와 일치 |
| 운영 반영 | 예시 IP·인증서·DB VIP만 실환경 교체 |
| 선택 | Gateway / OC / UJ는 기본 세트 밖 |

---

## 2. 이 패키지가 하는 일

1. 단일 도메인에서 Apache SSL + Sticky 라우팅  
2. Tomcat 클러스터에 ARC05 WAR 동일 배포  
3. DeltaManager로 센터 내부 세션 유지  

```text
Client → L4 → Apache → Tomcat×3 → /sv /eb … WAR → DB
```

---

## 3. Context / WAR (SoT)

| WAR | Context | Module |
|-----|---------|--------|
| ic.war … mg.war | `/ic` … `/mg` | `*-service` |
| om.war | `/om` | tcf-om |
| batch.war | `/batch` | tcf-batch |
| ui.war | `/ui` | tcf-ui |
| jwt.war | `/jwt` | tcf-jwt |

- `/` → `/ui/`  
- Health: `/sv/actuator/health`  
- Online: `POST /sv/online`  

상세: [04-inventory-셋팅정보.md](./04-inventory-셋팅정보.md)
