## 구조

```text
03-spring-boot/
├── gradle-bootWar-fragment.gradle   # JDK 21 + bootWar (권장)
├── pom-war-fragment.xml             # 레거시(Maven) — 사용 금지 권장
└── src/main/
    ├── java/.../session|config/
    ├── resources/application*.yml
    └── webapp/WEB-INF/web.xml       # <distributable/>
```
