# NSIGHT Engine — 아키텍처 셋팅정보 (인덱스)

> 읽는 순서: **03** / 전체: [01-README.md](./01-README.md)  
> 대상: Apache + 외장 Tomcat(DeltaManager) + Spring Boot WAR (ARC05)  
> **수치·표 SoT는 `04`~`10` 계층별 셋팅정보이다.** 이 문서는 개요·링크만 유지한다.

---

## 1. 논리 구성

```text
Client → L4/GSLB → Apache (SSL+Sticky) → Tomcat×3 (DeltaManager)
                                      → core12(+jwt) WAR → DB
```

| 원칙 | 내용 |
|------|------|
| Context | `/ic` `/sv` `/eb` … (ARC05) |
| 동일 배포 | 전 Tomcat 노드에 동일 WAR 세트 |
| Sticky | `JSESSIONID` route ≡ `jvmRoute` |
| DeltaManager | **센터 내부만** |
| Timeout | 거래는 짧게, 세션은 길게 |
| JDK | **21** · 빌드 Gradle `bootWar` |
