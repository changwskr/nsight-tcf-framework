아래처럼 NSIGHT TCF Framework 개발자 가이드 목차를 잡는 것이 좋습니다.기존 Java 코딩 스타일 가이드는 Spring Boot, MyBatis, 예외, 로그, 트랜잭션, 보안, 테스트 기준을 이미 포함하고 있고, TCF 설계 기준은 HTTP/JSON 표준 전문 + ServiceId Dispatcher + Handler → Facade → Service → Rule → DAO/Mapper 구조를 중심으로 합니다.
