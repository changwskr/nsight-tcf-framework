## DB 역할 (19.3)

| DB | 용도 |
|----|------|
| **RDW** | 실시간 조회, Single View |
| **ADW** | 분석·집계 |
| **SESSIONDB** | Spring Session, TCF_USER_SESSION |
| **OMDB** | 사용자, Catalog, 통제, 코드 |
| **LOGDB** | TCF_TX_LOG, OM_AUDIT_LOG |
| **TCFDB** | (논리) 통제·Timeout·Idempotency |
| **FILEDB** | 파일 메타, 다운로드 이력 |
| **JWTDB** | Token, Refresh, Denylist |

## TCF 통제 테이블 (19.5)

| 테이블 | 역할 |
|--------|------|
| OM_SERVICE_CATALOG | serviceId 마스터 |
| TCF_TRANSACTION_CONTROL | Allow-List |
| TCF_SERVICE_TIMEOUT_POLICY | Timeout |
| TCF_IDEMPOTENCY_KEY | 중복 방지 |
| TCF_SERVICE_AUDIT_POLICY | 감사 정책 |
