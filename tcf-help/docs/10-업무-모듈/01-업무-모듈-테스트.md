# 업무 모듈 테스트

[홈 화면 업무 모듈](/index.html#moduleSection)에서 업무코드별 테스트 JSON을 편집하고 Relay로 전송합니다.

## 흐름

1. 대상 WAS bootRun (예: SV 8086)
2. tcf-ui bootRun (8099)
3. `/sv/index.html` 등에서 serviceId·요청 JSON 입력
4. 실행 → 응답 JSON 확인

## 규칙 요약

- URL: `POST /api/relay/{code}/online`
- `header.serviceId`가 Catalog에 등록되어 있어야 함
- Controller를 업무에서 만들지 않음 — `tcf-web` 공통 `/online` + Handler
