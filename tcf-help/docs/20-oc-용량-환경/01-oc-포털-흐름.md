# OC 포털 흐름

[OC 포털](/oc/index.html)은 **용량 산정(CAP)** 과 **환경설정(ENV)** 화면으로 이동하는 허브입니다.

## 권장 순서

1. [용량 산정](/oc/capacity.html) — CAP-010~050
2. [ENV-002](/oc/env-002.html) — 조건 입력 · **산정 실행**
3. [ENV-003](/oc/env-003.html) — TPS·VM 결과
4. [ENV-004](/oc/env-004.html) — 계층별 설정 Grid
5. [Rule 점검](/oc/rule-check.html) — 설정 업로드·Rule Engine
6. [종합 보고서](/oc/check.html) — 최종 검토

## API

- Relay 경로: `/api/oc/env/...`, `/api/oc/capacity/...`
- 백엔드: `tcf-oc` (bootRun 8094)
