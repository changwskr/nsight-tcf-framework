# ENV-002 ~ ENV-004

환경설정 점검은 **ENV-002 산정 실행** 결과를 기준으로 이어집니다.

## ENV-002 — 산정 실행

- 경로: `/oc/env-002.html`
- 지점·사용자·VM 프로파일·동시 요청률 등 입력
- **「산정 실행」** 으로 플래너 결과를 브라우저 캐시에 저장

> 산정 없이 ENV-003/004를 열면 「산정 결과 없음」 안내가 표시됩니다.

## ENV-003 — TPS·VM

- 경로: `/oc/env-003.html`
- 시나리오별 TPS·TPMC·필요 VM·DB Pool 표
- Core당 TPS 선정·가이드 요약

## ENV-004 — 계층별 설정

- 경로: `/oc/env-004.html`
- UI → GSLB → L4 → … → MyBatis 스택 탭
- 권장값 vs 현재값 Grid·JVM 사이징
