# 로컬 기동·포트

|------|------|---------|------|

## OC + UI 함께 쓸 때

1. `tcf-oc:bootRun` (8094) — API
2. `tcf-ui:bootRun` (8099) — 화면·Relay

UI는 `/api/oc/...`를 Relay로 tcf-oc에 전달합니다. **8099만** 켜고 OC API가 없으면 ENV·CAP 화면이 빈 상태일 수 있습니다.

## WAR 배포 (ztomcat)

Tomcat 통합 배포 시 context가 `/ui`, `/sv` 등으로 분리됩니다. URL은 `ui-context.js`가 `/ui` 접두사를 보정합니다.
