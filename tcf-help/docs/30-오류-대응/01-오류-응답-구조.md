# 오류 응답 구조

업무·Relay 응답의 표준 오류 필드입니다. 화면 알림 **HELP** 버튼은 이 정보를 [오류 상세](/help.html#errors)에 기록합니다.

## result / error 영역

| 필드 | 설명 |
|------|------|
| resultCode | `SUCCESS` 또는 오류 코드 |
| errorCode | 업무·시스템 오류 코드 |
| errorMessage | 사용자 메시지 |
| errorDetail | 예외 스택·상세 |
| errorSystemId | 시스템 ID |
| errorDateTime | 발생 시각 |

## Relay 실패 시 추가

- `httpStatus` — HTTP 상태
- `serviceId` — 요청 serviceId
- `traceId` — 추적 ID
- `targetUrl` — 중계 대상 URL

## 확인 방법

1. 업무 화면에서 오류 발생
2. 알림 **HELP** 클릭 → `/help.html#errors`
3. 또는 [오류 상세](/help.html#errors)에서 샘플 버튼으로 UI 확인
