# NSIGHT TCF 문서 지도

`nsight-tcf-framework` 저장소의 **전체 마크다운 문서**를 HELP **문서 라이브러리**에서 검색·열람할 수 있습니다.

## HELP 화면 구성

| 화면 | URL | 용도 |
|------|-----|------|
| HELP 홈 | `/help.html` | 운영 가이드 요약 · 오류 상세 |
| 문서 라이브러리 | `/help/library.html` | **전체 600+ 문서** 검색·목록 |
| 문서 뷰어 | `/help/view.html?src=경로` | 원문 마크다운 읽기 |
| 운영 가이드 | `/help/view.html?doc=...` | HELP 전용 짧은 가이드 |

## 코퍼스 — 무엇부터 읽을지

| 순서 | 코퍼스 | 대상 독자 | 설명 |
|------|--------|-----------|------|
| 1 | **HELP 운영 가이드** | UI 사용자 | Relay, OC, 오류 — [라이브러리에서 필터](library.html?corpus=help-native) |
| 2 | **zguide** | 개발자 | 모듈별 bootRun·Handler 시작 — [zguide](library.html?corpus=zguide) |
| 3 | **zman** | 설계·개발 | 설계서↔코드 대조 — [zman](library.html?corpus=zman) |
| 4 | **zdoc** | 개발·운영 | 세션, SSO, 페이징 — [zdoc](library.html?corpus=zdoc) |
| 5 | **znsight-man** | 개발·QA | 표준, ServiceId, 테스트, 배포 — [znsight-man](library.html?corpus=znsight-man) |
| 6 | **용량·환경** | 아키텍트 | TPS, VM, Timeout — [capacity](library.html?corpus=capacity) |
| 7 | **아키텍처** | 아키텍트 | docs, zarchitecture — [architecture](library.html?corpus=architecture) |
| 8 | **TCF 교재** | 입문자 | ztcfbook / -m / -h — [교재](library.html?corpus=ztcfbook) |
| 9 | **모듈 README** | 개발자 | tcf-*, *-service — [README](library.html?corpus=module-readme) |

## 문서 갱신

```bash
node tcf-help/_scripts/scan-repo-docs.mjs
node tcf-help/_scripts/mirror-library.mjs
gradle :tcf-help:exportHelp :tcf-ui:processResources
```

카탈로그는 저장소 스캔 시 자동 생성됩니다 (`doc-catalog.json`).
