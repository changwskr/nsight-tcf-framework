# HELP 운영 가이드

`tcf-help/docs/` — 브라우저 HELP에서 바로 읽는 **짧은 운영·화면 가이드** (12편).

## 포함 주제

- TCF UI · Relay · 로컬 포트
- 업무 모듈 테스트 · OM Admin
- OC 용량 산정 · ENV-002~004 · Rule 점검
- 오류 응답 · Relay 실패

## vs 저장소 다른 문서

| 구분 | HELP docs | zguide / znsight-man |
|------|-----------|----------------------|
| 분량 | 1~2 화면 | 수십~수백 페이지 |
| 목적 | UI 사용·빠른 시작 | 개발 표준·심화 |
| URL | `?doc=...` | `?src=tcf-help/docs/...` 또는 라이브러리 |
